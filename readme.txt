required packages:
  pyhon3
  pytomplegui
  colorama
  pickle
  socket
  
  Chat server and app with encryption
  
To start the chat server execute : "python3 server.py" from a terminal in the chat directory. 

To start the Chat app client with a GUI (graphical interface) execute: "python3 chat_start.py" from a terminal in the chat directory. 
You will need to know the chat servers IP address.
